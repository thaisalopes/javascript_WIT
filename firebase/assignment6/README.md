Assignment 6 – Intermediate JavaScript & Firebase

This project allows users to sign up and log in using Firebase Authentication.

Features:
- User sign-up with name, email, password, and date of birth
- Login and logout functionality
- Stores user data in Firebase Realtime Database
- Calculates how many days are left until the user’s birthday and displays on screen unless it's the user's birthday
- Displays a birthday message and a random quote on user’s birthday
